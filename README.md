# KelsonCraft Pack 1.21.1

There is a lot of mods that I enjoy in this pack, I have not yet published it but when I do, it'll be on CurseForge.

I am still making changes to this mod pack, I'll release it with an alpha version for now, I may add new mods or remove them so this mod list isn't final yet.

This is going to be the KelsonCraft mod pack for 1.21.1 on NeoForge, which has a lot of power mods, tech mods and stuff like that.

# Mod pack Info

This mod pack has Create, and Mekanism, also a lot of the Macaw's Mods.
It uses Sodium, Lithium, Entity Culling, and a lot of the performance mods to try to keep the game running well.

I have most of the YUNG's better world generation mods in this pack.


The minimap has been moved to the right side of the screen where I use FTB Ultimine, the minimap is [Xaero's minimap](https://www.curseforge.com/minecraft/mc-mods/xaeros-minimap) and [Xaero's worldmap](https://www.curseforge.com/minecraft/mc-mods/xaeros-world-map).


You can open double doors at the same time by opening them normally with the [Double Doors mod](https://www.curseforge.com/minecraft/mc-mods/double-doors).


Climbing ladders up and down is much faster with the [Faster Ladder Climbing](https://www.curseforge.com/minecraft/mc-mods/faster-ladder-climbing) mod, look up or down while on a ladder and you will climb faster.


This pack has [Waystones](https://www.curseforge.com/minecraft/mc-mods/waystones) for traveling around the map easier, and the [Moving Elevators](https://www.curseforge.com/minecraft/mc-mods/moving-elevators) mod has elevators with displays that can be crafted.


Most of the FTB mods are in this pack, you can protect your land with [FTB Chunks](https://www.curseforge.com/projects/314906), run essential commands like `/rtp` with [FTB Essentials](https://www.curseforge.com/minecraft/mc-mods/ftb-essentials).


I don't have FTB Quests setup yet but it is installed.
You can use [FTB Ultimine](https://www.curseforge.com/minecraft/mc-mods/ftb-ultimine-forge) for a Vein Miner with the `~` key


The storage mods in this pack are [Functional Storage](https://www.curseforge.com/minecraft/mc-mods/functional-storage), [Iron Chests](https://www.curseforge.com/minecraft/mc-mods/iron-chests),  


Attack cool downs are pretty much disabled in this pack.

I have a script that disables farmland crop trampling, so you cannot step on farmland to break it or other mobs cannot.

# Notable mods in this pack
* AE2 - Applied Energistics 2
* Cooking for Blockheads
* Create and a bunch of addons for it.
* CC Tweaked - Computer Craft mod
* Farmer's Delight
* Farming for Blockheads
* Easy Villagers - Shift and left click to pick up villagers, and make iron farms, and other farms with them.
* e4mc - Open LAN game to public with a link.
* Iron Furnaces - Faster furnaces, can also be powered by Forge Energy.
* Jade - Display what a block is on screen.
* JEI - Item and recipe viewing mod
* Metal Barrels - additional tiers for barrels.
* Mob Catcher - Catch mobs and bring them back easily
* Mob Grinding Utils - Mob farming utilities
* Pipez - Adds energy pipes, item pipe, gas pipe, and universal pipes.
* Mekanism
* Ping Wheel - Ping a spot in game with the extra mouse buttons or a specified keybind.
* Refined Storage - Storage management mod which uses Forge Energy.
* SecurityCraft - Security locked doors, security cameras and more.
* Sign Copy - Copy and paste texts onto signs.
* Simple Voice Chat - Voice chat in-game
* Trash Cans - Add trash can items for piping into.
* WorldEdit - This is only enabled in creative.
